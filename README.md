# Workout Builder 💪

Application PWA de construction et d'analyse de programmes de musculation.

## Déploiement sur Vercel (5 étapes)

1. Crée un compte sur [github.com](https://github.com) si tu n'en as pas
2. Crée un nouveau repository GitHub, nomme-le `workout-builder`, et uploade tous ces fichiers dedans
3. Crée un compte sur [vercel.com](https://vercel.com) en te connectant avec GitHub
4. Sur Vercel : "Add New Project" → sélectionne le repository `workout-builder` → "Deploy"
5. C'est en ligne ! Ouvre l'URL sur Safari iOS → icône Partager → "Sur l'écran d'accueil"

## Installation sur iPhone

1. Ouvre l'URL de ton app dans **Safari** (pas Chrome)
2. Appuie sur l'icône **Partager** (carré avec flèche vers le haut)
3. Fais défiler et appuie sur **"Sur l'écran d'accueil"**
4. Appuie sur **Ajouter**

L'app apparaît comme une vraie application sur ton écran d'accueil !
